# BackendSISED
The aim of this project is to set up an integrated system for monitoring the equivalence of diplomas inside  the Unit to Support the Information of Fiscal and Financial Services. the app is fullstack ,this part is the backend in Spring Boot 2.1.5 Link to the frontend repository in Angular 8 https://github.com/ckechi1/FRONTENDSISED
